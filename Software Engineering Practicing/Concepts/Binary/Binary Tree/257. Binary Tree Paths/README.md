## 257. Binary Tree Paths

```Tag```: ```Binary Tree``` ```Depth-First Search``` ```Backtracking```

#### Difficulty: Easy

Given the ```root``` of a binary tree, return _all root-to-leaf paths in __any order___.

A __leaf__ is a node with no children.

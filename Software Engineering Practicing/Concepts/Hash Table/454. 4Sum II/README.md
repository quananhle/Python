## 454. 4Sum II

```Tag```:

#### Difficulty: Medium

Given four integer arrays ```nums1```, ```nums2```, ```nums3```, and ```nums4``` all of length ```n```, return _the number of tuples ```(i, j, k, l)```_ such that:

- ```0 <= i, j, k, l < n```
- ```nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0```

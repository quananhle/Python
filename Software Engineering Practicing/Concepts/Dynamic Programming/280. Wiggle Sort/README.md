## 280. Wiggle Sort

```Tag```: ```Greedy Algorithm``` ```Sorting``` ```Dynamic Programming```

#### Difficulty: Medium

Given an integer array ```nums```, reorder it such that ```nums[0] <= nums[1] >= nums[2] <= nums[3]....```

You may assume the input array always has a valid answer.
